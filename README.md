# LooTTGen

**LooTTGen** — простой скрипт для генерации лута, созданный с использованием **Tkinter** и поддержкой **SQLite**.  

## Возможности
- Создание и редактирование предметов.
- Формирование таблиц лута с настройкой шансов выпадения, уникальности и обязательного появления предметов.
- Генерация лута для пользовательских контейнеров на основе созданных таблиц.  

## Инструкции по использованию

### 1. Добавление предметов
1. Перейдите на вкладку **Items**.
2. Нажмите кнопку **Добавить предмет**.
3. Введите необходимые характеристики предмета.

### 2. Создание таблиц лута
1. Перейдите на вкладку **Таблицы лута**.
2. Нажмите **Новая таблица**, чтобы создать список предметов.
3. Выберите таблицу в левом окне.
4. Нажмите **Добавить предмет** и выберите нужный предмет из списка.
5. Настройте:
   - **Шанс выпадения** — отражает редкость предмета.
   - **Уникальность** — предмет не может появляться несколько раз в одной генерации.
   - **Обязательное появление** — предмет добавляется в генерацию сверх максимального количества предметов.

> Свойства **Уникальное** и **Обязательное** можно изменить напрямую в таблице, дважды кликнув по соответствующему полю.

### 3. Настройка контейнеров
1. Перейдите на вкладку **Контейнер**.
2. Создайте контейнер и привяжите к нему одну из созданных таблиц.
3. Выберите контейнер из списка и проведите генерацию предметов.

## Требования
- Python 3.x
- Tkinter
- SQLite
